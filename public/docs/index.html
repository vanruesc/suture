    Suture

    A 3D game engine that stitches several systems together in order to provide a single API for game development purposes.

    This project is currently on hold.

    API Reference

    Installation

    npm install suture

    Usage

    import { Suture } from "suture";

const suture = new Suture();
suture.run();
